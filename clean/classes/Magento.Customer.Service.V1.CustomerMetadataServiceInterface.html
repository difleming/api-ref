\Magento\Customer\Service\V1

\Magento\Customer\Service\V1CustomerMetadataServiceInterface

Defines constants and required methods for customer metadata.

The constants for customer metadata define attribute IDs and entity types for customer and customer address.

The methods for customer metadata enable users to:

  • Show attribute metadata or custom attribute metadata for a customer or customer address.
  • List EAV attribute metadata for a specified entity type.
  • List attributes for a specified entity type filtered by a form code.
  • List attribute metadata for all customers or customer addresses.
