\Magento\Customer\Service\V1

\Magento\Customer\Service\V1CustomerAddressService

Use customer address service objects to create and manage customer addresses.

Use the methods on customer address service objects to:

  • Add, update, or validate a set of addresses.
  • List, show information for, and delete addresses.
  • Show the default billing address or shipping address for a specified customer.

saveAddresses()

saveAddresses( $customerId,  $addresses) : array<mixed,integer>

Insert and/or update a list of addresses.

This will add the addresses to the provided customerId. Only one address can be the default billing or shipping so if more than one is set, or if one was already set then the last address in this provided list will take over as the new default.

This doesn't support partial updates to addresses, meaning that a full set of data must be provided with each Address
